# 🌌 GlobalWay DApp

Децентрализованная платформа для глобальных интернет-возможностей на базе opBNB блокчейна.

## 🚀 Функции

- **Многоуровневая матричная система** (12 уровней)
- **Партнерская программа** с автоматическим распределением
- **Токен GWT** с механизмом роста цены
- **Инвестиционные пулы** и лидерские программы
- **Полная децентрализация** без KYC
- **Мобильная оптимизация** PWA
- **Мультиязычность** (EN/UK/RU)

## 🛠 Технологии

- **Blockchain**: opBNB (оптимизированная BSC)
- **Smart Contracts**: Solidity ^0.8.19
- **Frontend**: Vanilla JavaScript + Web3.js
- **CSS**: Космический дизайн с адаптивностью
- **Wallet**: SafePal, MetaMask совместимость
